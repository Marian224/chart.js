# 📊 Data Visualization Dashboard

Простий та елегантний веб-інтерфейс для візуалізації статистичних даних. Проєкт демонструє інтеграцію бібліотеки **Chart.js** для створення інтерактивних графіків.

## 🚀 Огляд проєкту

Цей проєкт створений для наочного відображення бізнес-показників (продажів та бюджету) на одній сторінці. Він поєднує в собі мінімалістичний дизайн та високу інформативність.

### Основні функції:
* **Лінійний графік (Line Chart):** Відображає динаміку продажів за місяцями (від січня до липня) з плавними лініями.
* **Кільцева діаграма (Doughnut Chart):** Наочно демонструє розподіл щомісячного бюджету за категоріями (оренда, їжа, транспорт тощо).
* **Інтерактивність:** Графіки реагують на наведення курсору, відображаючи точні значення у підказках (tooltips).
* **Адаптивний дизайн:** Завдяки Flexbox та налаштуванням Chart.js, сторінка коректно відображається на різних розмірах екранів.

## 🛠 Технологічний стек

* **HTML5** — структура сторінки.
* **CSS3** — стилізація (використано шрифт Segoe UI, м'які тіні та заокруглені контейнери для сучасного вигляду).
* **JavaScript (ES6)** — логіка обробки даних.
* **Chart.js v4.4.2** — потужна бібліотека для рендерінгу графіків через Canvas API.

## 📁 Структура файлів

* `index.html` — основна розмітка документа.
* `style.css` — опис візуального стилю та сітки.
* `script.js` — ініціалізація та конфігурація графіків.

## ⚙️ Як запустити

1. Скопіюйте файли проєкту на свій комп'ютер.
2. Відкрийте файл `index.html` у будь-якому сучасному браузері.
3. Переконайтеся, що у вас є підключення до інтернету (для завантаження бібліотеки Chart.js з CDN).
